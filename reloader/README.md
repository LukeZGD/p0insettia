# p0insettia semi-untether
A tool for semi-untethered jailbreak of iOS 10.3.4 iPhone 5 with socket exploit.  

## Note
- All at your own risk!  

## Supported devices
- iPhone 5 (N41/N42) - iOS 10.3.4
